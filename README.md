# kelompok-wira_gian

Tugas dikerjakan secara berkelompok 2 anggota, boleh 3 anggota (maks) kalau satu orang tidak ada timnya.
Anggota 1 : wira
Buat repo baru, nama kan sesuai nama kelompok
Tambahkan kolaborator repo untuk teman tim
Buat branch dengan format penamaan (nama/nim)
Lalu isi file bebas kedalam branch itu. + push
Anggota 2 : gian
Clone repo
Buat branch baru didalam repo dengan format penamaan (nama/nim)
Lalu isi file bebas kedalam branch itu. + push
Anggota 3 (jika ada)
Sama dengan orang ke 2
Buat laporan Langkah Langkah yang dilakukan + dokumentasi (screenshot).

Boleh mengumpulkan laporan secara perwakilan.
